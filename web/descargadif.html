Calibre para windows
Gestor y conversor de libros electrónicos
Calibre es un gestor y conversor de ebooks gratuito con el que podrás administrar estos archivos en tu ordenador Windows de manera sencilla y cómoda. Se trata por tanto de un programa similar a Alfa eBooks Manager o Lucidor, aunque posee una mayor variedad de funciones que la gran mayoría de aplicaciones del segmento. De este modo, Calibre te permitirá una completa organización y gestión de tus libros electrónicos, pudiendo ordenarlos en tu biblioteca virtual, modificar sus características, compartirlos con otros dispositivos o transformarlos a múltiples formatos. Relativo a este último aspecto, el software es compatible con ePub, HTML, PDF RTF, txt, MOBI u ODT, entre otros.
